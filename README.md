# Number-System-Converter-in-C-Programming-Language
The C programming language implements a lightweight Number System Converter which operates through a menu system. The program helps students and developers learn basic base conversion techniques. The program enables users to enter a decimal number in Base-10 format which the system converts to Binary, Octal, and Hexadecimal formats.
